# Google Search Console & Local SEO Setup Guide

## 1. Google Search Console Submission

### Step 1: Add Property
1. Go to [Google Search Console](https://search.google.com/search-console)
2. Click "Start now" or "Add property"
3. Choose URL prefix: `https://thelinkagedigital.com`
4. Click "Continue"

### Step 2: Verify Domain
You can verify using:
- **HTML file**: Download HTML file and upload to root
- **HTML tag**: Copy meta tag and add to index.html
- **Google Analytics**: If you have GA tracking enabled
- **Google Tag Manager**: If you use GTM

### Step 3: Submit Sitemap
1. In Search Console left sidebar, go to "Sitemaps"
2. Enter sitemap URL: `thelinkagedigital.com/sitemap.xml`
3. Click "Submit"
4. Wait for Google to crawl (can take 24-48 hours)

### Step 4: Monitor Coverage
1. Go to "Coverage" section
2. Check for any errors or warnings
3. Fix any indexation issues
4. Monitor the "Excluded" tab for pages Google didn't index

### Step 5: Check Performance
1. Go to "Performance" tab
2. Monitor:
   - Clicks from search results
   - Impressions
   - Click-through rate (CTR)
   - Average position
3. Track rankings for your target keywords

## 7. Monitor and Measure

### Monthly Reporting
Track these metrics:
- **Organic traffic**: Sessions from search
- **Keyword rankings**: Position for target keywords
- **Click-through rate (CTR)**: From search results
- **Impressions**: How many times shown in search
- **Conversions**: Contact form submissions from search

### Tools for Tracking
- **Google Analytics**: Traffic and behavior
- **Google Search Console**: Search performance
- **SEMrush**: Keyword tracking
- **Ahrefs**: Backlink and ranking monitoring
- **Google Ads**: See search volume for keywords

### Google Analytics Setup (if not done)
1. Add Google Analytics tracking to website
2. Set up goals:
   - Contact form submission
   - Phone call clicks
   - Email clicks
   - Service page visits
3. Create custom dashboards for quick reference
